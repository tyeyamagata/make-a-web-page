Lesson 1: Basics of the Web and HTML

What is 'The Web?'

The Web is network of computers sharing information. When your browser displays information on your screen, it's showing you the HTML file that a server sent via HTTP request.

What is HTML?

Hypertext Markup Language, or HTML, is a language used to create documents that control "what you see" and "how it looks" on the web.

What are "Tags" and "Elements?"

Elements are the structure and content of what you see on a web page. Tags are used to define the elements to correctly apply properties and style. More information about tags and elements can be found here.

What are "Inline" and "Block" elements?

Inline elements are usually text elements that are not effected by breaks or indentation unless specified. Block elements are separated by "invisible boxes" to give them space around other elements. More information about inline and block elements can be found here.

Lesson 2: HTML Document Structure

What tools do Web Developers use?

HTML, CSS, and Javascript are 3 basic tools commonly used to construct web pages. HTML is the language of the Web with a set of specific syntax and rules. CSS is the program used to add style. Style can affect the position, size, and color of elements. Finally, Javascript can change HTML content, HTML attributes, CSS style, and can validate input.

What is Webpage Structure?

Structure is used to organize code to make it easier for humans to understand and read. First of all it is important to note that not all elements are visible from the browser. Some elements are used to help read the code like a map. All elements are, however, rectangular. Everything you see in a webpage is contained within a box indicating the element's boundaries. At first glance structure can be very simple. It is possible to dig deeper into the layers of each webpage element and find that the structure branches off much like a tree.

Lesson 3: The Fun Begins with CSS! Style that web page!

What is "The Box Model?"

The Box Model is used in developer tools to visually display an element and its dimensions. It consists of margins, borders, padding, and the content of the element.

What is the most important thing to avoid when writing code?

Avoid repetition! Errors most frequently occur in repeated instances of the same code where minor changes are made. If you're not consistant, those errors can lead to malfunctioning code. It's best to keep things as simple as possible to avoid these mistakes.

What is CSS?

Cascading Style Sheets, or CSS, control the style of HTML elements. By using tags, the CSS gives style to all elements of the same class-type.
